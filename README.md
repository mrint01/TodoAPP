# ToDo App ⚡️
# Using MEAN stack: Angular ♥ ExpressJs ♥ MongoDB ♥ Nodejs ♥

### After clone project
# For backend
enter directory:
```
cd server
```
install dependencies:
```
npm install
```
run server:
```
npm run start-server
```


# For Frontend
enter directory:
```
cd client
```
install dependencies:
```
ng install
```
run frontend app
```
ng serve
```
